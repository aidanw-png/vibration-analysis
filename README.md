# Vibration Analysis — CWRU Bearing Fault Diagnostics

A Python portfolio project demonstrating signal processing techniques for
rotating machinery fault detection using the Case Western Reserve University
(CWRU) bearing dataset. This project covers FFT spectral analysis, Welch's
power spectral density estimation, envelope analysis, octave-band filtering,
and automated PDF report generation.
